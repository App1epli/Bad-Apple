# 🍎 bab-apple

A small experimental project inspired by the legendary **Bad Apple** animation.

This repository explores rendering, media processing, and creative visualization — recreating the iconic monochrome style of the Bad Apple video using custom code.

## 🎬 About

The original Bad Apple animation became an internet phenomenon thanks to its minimalistic black-and-white silhouette style.  
Because of its simple visuals, developers often recreate it on unusual hardware or inside software projects.

This project is my own experimental implementation.

## ⚙️ Features

- 🖤 Monochrome / silhouette rendering
- 🎥 Frame-based animation
- 🧪 Experimental visual pipeline
- 💻 Built as a personal coding playground
